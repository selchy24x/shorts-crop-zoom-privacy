Shorts Crop Zoom Privacy Policy

Last updated: June 2, 2026

Shorts Crop Zoom is a Chrome extension that uses a bundled local AI model to detect and enlarge the horizontal content area inside YouTube Shorts.

Summary

Shorts Crop Zoom processes the currently visible YouTube Shorts video frame locally in the user's browser when the user activates the extension. This local processing is used only to detect the crop area and display the enlarged overlay.

The extension does not transmit, sell, share, or store user data on external servers.

Data Handled

The extension handles website content locally: specifically, the currently visible video frame from a YouTube Shorts page selected by the user. This is processed only in browser memory for the extension's single purpose.

The extension does not collect or transmit the following categories of data:

  • Personally identifiable information
  • Health information
  • Financial or payment information
  • Authentication information
  • Personal communications
  • Location information
  • Web history
  • User activity logs

Local Processing

When the user activates the extension on a YouTube Shorts page, the extension reads the currently visible video frame in the browser, runs local AI inference using bundled extension files, and displays a live crop overlay.

Video frames are processed locally in the user's browser. They are not uploaded to any server and are not saved by the extension.

Network Requests

The extension does not send analytics, telemetry, usage logs, video frames, URLs, identifiers, or browsing data to remote servers.

The AI model and runtime files are bundled with the extension and loaded from the extension package. The extension does not download remote code or remote model files.

Permissions

The extension uses the minimum permissions needed for its single purpose:

  • activeTab: used when the user activates the extension for the current YouTube Shorts tab.
  • scripting: used to inject the bundled runtime and content script into an already-open Shorts tab when necessary.
  • YouTube Shorts page access: used to detect the visible video element and draw the local zoom overlay only on YouTube Shorts pages.

Third Parties

Shorts Crop Zoom does not sell, transfer, or disclose user data to third parties.

This extension is not affiliated with YouTube or Google.

Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Changes

If this privacy policy changes, the updated version will be published on this page.
